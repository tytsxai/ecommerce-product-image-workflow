经理一键生成（操作卡片）

1) 你要做的只有 4 步

  1. 上传供应商产品图（多张）。
  2. 选择风格包（Style Pack）：Minimal minimal_white / Lifestyle lifestyle_warm / Premium premium_dark（或预设）。
  3. （可选）上传个性化素材：照片/文字/Logo。
  4. 点击“一键生成” → 自动输出：showcase + spec + howto。

推荐默认选项见：16-默认配置-一键生成.md。

2) 若只有产品照片（无任何文字）

  • 先填： 09-输入采集表单-经理填写.md 里的 C1（补齐规格与步骤的英文文本）。
  • 再一键生成。

如果经常遇到产品变形，先对照 17-供应商图最小要求.md 向供应商补图。

3) 一分钟质检（通过/退回）

任意一条不满足就退回。

  • 产品本体不变：形状/结构/颜色/比例/关键细节一致。
  • 背景明显不同：场景/光线/构图/道具/色调至少多项变化（不是复刻供应商图）。
  • 规格图：数值/单位/含义完全一致；字体不同；背景不同。
  • 说明图：含义/步骤一致（可轻改写英文）；字体不同；背景不同。
  • 图片上所有文字必须为英文。

4) 退回时怎么选原因

  • 产品变形/缺结构 → product_changed
  • 背景太像供应商图 → background_too_similar
  • 文案非英文/拼写错误 → text_not_english
  • 规格数值错误/单位错 → spec_value_error
  • 说明步骤含义变了 → howto_meaning_changed
  • 个性化规则不符合 → personalization_rule_violation

详细退回处理见 15-常见问题与退回处理.md。

5) 生成后留痕（建议 30 秒）

  • 填写一条记录：19-生成批次记录-经理填写.md
