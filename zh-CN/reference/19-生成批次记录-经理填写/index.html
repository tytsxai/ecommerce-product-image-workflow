记录模板（每批复制一份）

  • Date:
  • Batch ID（自定义，如 2025-12-22-A）：
  • ProductID / SKU：
  • Supplier input link/path（weiyun/本地）：
  • Supplier images filenames（可选）：
  • Style Pack：Minimal minimal_white / Lifestyle lifestyle_warm / Premium premium_dark / Other
  • Background variant（可选，同风格包第几套/ID）：
  • Output set：Minimum / Extended
  • Personalization：None / Photo / Text / Logo（可多选）
  • Text source（可选）：spec_source=OCR/表单/数据库；howto_source=OCR/表单
  • Versions（可选）：workflow_version / template_version / stylepack_version（约定见 13-版本与变更记录.md）
  • QC result：Pass / Reject
  • Reject tag（如 Reject 才填）：product_changed / background_too_similar / spec_value_error / howto_meaning_changed / text_not_english / personalization_rule_violation / other
  • Notes（可选）：
