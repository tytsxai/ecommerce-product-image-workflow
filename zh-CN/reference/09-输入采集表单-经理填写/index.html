输入采集表单（经理填写）

A) 基础信息（必填）

  • ProductID / SKU：
  • Product Name (EN)：
  • Category（可选）：
  • Supplier images：上传（多张）

    若供应商图质量不佳导致经常退回，可先参考 17-供应商图最小要求.md 补图。

A1) 供应商图包含哪些内容（必填勾选）

用于判断是否需要你补填规格/说明文本。

  • [ ] 仅产品照片（无任何文字）
  • [ ] 含规格/尺寸图（有数值/单位）
  • [ ] 含使用说明/步骤图（有步骤/注意事项）
  • [ ] 含卖点文案/信息图（有文字说明）

B) 生成选择（必填）

  • Style Pack：Minimal minimal_white / Lifestyle lifestyle_warm / Premium premium_dark / Other:（参考 11-风格包-StylePacks.md）
  • Output set：Minimum (3+2+2) / Extended:

C) 可选描述（有则填）

  • Short description（经理可用 EN 或 RU 输入）：
  • Must-have keywords（EN/RU）：
  • Must-avoid elements（EN/RU）：

    可复制的 EN/RU 指令模板见 12-经理指令模板-EN-RU.md。

C1) 若供应商图没有规格/说明文字（必填补齐）

只在你勾选了“仅产品照片（无任何文字）”时必填；否则留空即可（系统可从图里抽取，再由你复核）。

Specs（规格图文字源）

  • Units preference：cm / in（二选一）
  • Dimensions：L x W x H：
  • Specs list（至少 3 条）：
  • Spec 1:
  • Spec 2:
  • Spec 3:

How-to（说明图文字源）

  • How-to title（EN）：How to Use / How It Works / Instructions（任选其一或自填）
  • Steps（至少 3 步，每步英文短句）：
  • Step 1:
  • Step 2:
  • Step 3:
  • Tips/Notice（可选 2–4 条，英文短句）：
  • Tip 1:
  • Tip 2:

D) 个性化（按订单类型选择）

D1) Custom Photo

  • 是否需要：Yes/No
  • 素材来源：Customer upload / Internal library
  • 语义要求（EN/RU 输入）：例如 “couple photo, warm tone”

D2) Custom Text

  • 是否需要：Yes/No
  • 文案内容（EN 输出）：（内容需与供应商不同，但同一产品/同一批次保持同字体/字号）

D3) Custom Logo / Graphic

  • 是否需要：Yes/No
  • 素材：上传
  • 风格要求（EN/RU 输入）：例如 “minimal line icon”

E) 合规确认（必勾）

  • 我确认所上传的照片/Logo/图形具有商用授权或自有版权。
  • 我理解输出图片中文字必须为英文。
