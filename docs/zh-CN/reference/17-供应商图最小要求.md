# 供应商图最小要求

## 必须提供（最低）
- 至少 3 张清晰产品照片（同一款产品）：
  - 正面（主视角）
  - 45° 斜角（带厚度与结构）
  - 关键细节特写（接口/纹理/关键结构）
- 分辨率：单张边长 ≥ `1500px`（越大越好）
- 清晰度：不糊、不强压缩、不重水印遮挡产品关键区域

## 强烈建议（显著提升出图稳定性）
- 1 张白底或浅色棚拍主图（便于产品锁定与抠图）
- 若需输出规格图：提供包含尺寸/数值/单位的信息图（或 PDF/表格）
- 若需输出说明图：提供包含步骤/注意事项的说明图（或文字版说明）

## 不建议/尽量避免
- 复杂背景 + 产品与背景颜色接近（难分割、易改形）
- 强反光/镜面台面导致边缘不清晰
- 过度滤镜或极端阴影（影响产品材质与颜色判定）
