# 项目概览（Phase 1：图片）

## 一句话目标
经理上传供应商产品图（+可选描述/参考）→ 自动产出上架所需的「主图/场景图 + 规格图 + 使用说明图」，满足“产品一致、背景明显不同、文字规则可控、可商用”的验收标准。

## 范围与不范围
- 范围：图像再创作/重构（Image Re-composition），支持融合自定义文字/照片/Logo/图形；支持仅凭图片输入也可出结果。
- 不范围（另签 Phase 2）：任何视频生成、剪辑、动效、模板化视频批处理能力。

## 必须满足的硬约束（Non-negotiables）
- 产品本体 100% 一致：外观/结构/颜色/形状/比例不可变，真实感与比例合理。
- 背景必须明显不同：场景、光线、构图、道具、色调需变化；不得“同背景复刻”。
- 规格图：文本内容（数值/单位/含义）一致，但字体必须变化、背景必须变化。
- 说明图：含义一致（可轻改写），但字体必须变化、背景必须变化。
- 语言：经理输入（英语/俄语均可）；图片上所有文字必须为英语。

## 最小交付物（Deliverables）
- 一套可运行的图片生成工作流（面向非技术用户）。
- 一套可复用的模板/规则配置（背景、个性化内容、规格图、说明图）。
- 用示例产品集验证的输出资产（覆盖所有类别）。
- 使用说明（至少英文；可选俄文）。

## 角色与责任（建议）
- 经理：上传输入、选择风格/模板、发起生成、按质检清单做最终通过/退回。
- 运营/品牌：提供品牌色/字体偏好/禁用元素、定义风格包（Style Pack）。
- 供应链：确认“产品本体不可变”的判定口径（哪些部件算关键结构）。
- 实施方：搭建工作流、配置模板与质检规则、维护日志与版本。
